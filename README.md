# Website SMA Negeri 9 Cirebon

Website profil sekolah SMA Negeri 9 Cirebon yang menampilkan informasi seperti keunggulan, agenda, berita, dan SPMB.

## 📌 Fitur
- Beranda (Hero section dengan background)
- Keunggulan sekolah
- Agenda terdekat
- Berita terbaru
- Testimoni
- FAQ / SPMB

## 🚀 Cara Menjalankan
1. Download / clone project ini
2. Buka file `index.html` di browser
3. Pastikan file gambar (`img.jpg`) ada di folder yang sama

## 📁 Struktur File
- `index.html` → halaman utama
- `style.css` → styling tambahan

## 🛠️ Teknologi
- HTML
- CSS
